Create `src/app/hooks/useVoiceCapture.ts` — a React hook that captures microphone audio and streams it to OpenAI's Realtime API for transcription.

**Hook signature:**
```typescript
interface UseVoiceCaptureOptions {
  onSegment: (segment: TranscriptSegment) => void;
  onError?: (error: string) => void;
}

interface UseVoiceCaptureReturn {
  isConnected: boolean;
  isRecording: boolean;
  startRecording: () => Promise<void>;
  stopRecording: () => void;
  error: string | null;
}
```

**Implementation approach — OpenAI Realtime API via WebSocket:**

1. `startRecording()`:
   - Get API key from `import.meta.env.VITE_OPENAI_API_KEY` (will be set in .env)
   - Request microphone access via `navigator.mediaDevices.getUserMedia({ audio: true })`
   - Open WebSocket to `wss://api.openai.com/v1/realtime?model=gpt-4o-mini-realtime-preview`
   - Send `session.update` event to configure: `input_audio_transcription: { model: "gpt-4o-mini-transcribe" }`, `turn_detection: { type: "server_vad" }`, `modalities: ["text", "audio"]`
   - Use AudioContext + ScriptProcessorNode (or AudioWorklet) to read microphone PCM samples, convert to 16-bit PCM base64, send as `input_audio_buffer.append` events
   - Listen for `conversation.item.input_audio_transcription.completed` events — extract `transcript` field, create a `TranscriptSegment` with `crypto.randomUUID()` id and elapsed ms timestamp, call `onSegment()`

2. `stopRecording()`:
   - Send `input_audio_buffer.commit` then close WebSocket
   - Stop all MediaStream tracks
   - Close AudioContext
   - Set `isRecording = false`

3. Error handling:
   - If getUserMedia fails: set error "Microphone access denied"
   - If WebSocket fails: set error with message, call onError
   - Cleanup on unmount via useEffect return

**Audio format for OpenAI Realtime API:** PCM16 at 24kHz, mono, base64 encoded. Use AudioContext with sampleRate 24000. Read samples from processor, convert Float32 to Int16, base64 encode, send.

IMPORTANT: The OpenAI Realtime API key is sent as a header in the WebSocket URL protocol. Use: `new WebSocket(url, ['realtime', 'openai-insecure-api-key.' + apiKey, 'openai-beta.realtime-v1'])`.

Do NOT use the `Authorization` header approach (browsers don't support custom WebSocket headers). The subprotocol approach is OpenAI's documented browser method.
